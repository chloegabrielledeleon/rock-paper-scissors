# Rock Paper Scissors (Python)

## Overview
This project implements the classic Rock–Paper–Scissors game in Python.
It demonstrates basic programming concepts such as control flow,
user input handling, and randomization.

## Features
- User vs computer gameplay
- Random computer choices
- Input validation
- Score tracking

## Technologies Used
- Python
- Jupyter Notebook

## How to Run
Open the notebook and run all cells in order.
